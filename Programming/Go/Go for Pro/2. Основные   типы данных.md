Данные хранятся и используются в переменных, и все переменные Go должны иметь тип данных, который определяется явно или неявно.

### Тип данных error

Для представления условий ошибки и сообщений Go содержит специальный тип данных error. На практике это означает, что этот язык обрабатывает ошибки как значения.

Go следует такому соглашению о значениях `error`: если значение переменной error равно `nil`, то ошибки не было.

В качестве примера мы рассмотрим `strconv.Atoi()`, который используется для  преобразования значения `string` в `int`
