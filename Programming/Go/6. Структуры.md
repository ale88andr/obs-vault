`Struct` - нужен для объединения данных разных типов в один логически связанный объект.

`Struct` - основной способ создания пользовательских типов данных.

Основные цели использования
- объединяет несколько полей в одну сущность
- позволяет описывать сущности предметной области (например, `User`, `Product`)
- делает код более читаемым и структурированным
