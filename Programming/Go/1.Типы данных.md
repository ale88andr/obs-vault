**ТИП ДАННЫХ** - Множество значений и операций, которые могут быть применимы над этими значениями

**ПРИМИТИВНЫЕ ТИПЫ ДАННЫХ**
*Логические*
- bool
*Строковые*
- string
*Числовые*
- int8, uint8 (byte), int16, uint16, int32(rune), uint32, int64, uint64, int, uint, uintptr
- float32, float64
- complex
