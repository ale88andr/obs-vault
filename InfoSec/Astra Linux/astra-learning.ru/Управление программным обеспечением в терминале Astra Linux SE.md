Программное обеспечение для ОС Linux распространяется в программных пакетах. Пакет представляет собой архивный файл, содержащий программный код в бинарном или исходном виде, а также метаданные о программе, её версии, зависимостях и другую информацию.

Имя пакету присваивается в соответствии с соглашением и обычно имеет следующий вид:

﻿ ```
﻿ <имя_пакета>-<версия>-<архитектура>.deb
﻿ ```

Каждый программный пакет содержит только необходимый минимум файлов и библиотек, все остальные необходимые ему компоненты указываются в нём в качестве зависимостей. Зависимость — потребность программного пакета в ресурсах, которые находятся в составе других пакетов.
