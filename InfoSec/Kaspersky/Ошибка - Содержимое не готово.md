1. Откройте Kaspersky Security Center.
2. Перейдите в **Дополнительно** → **Хранилища**.
3. Откройте контекстное меню объекта **Обновление баз и программных модулей «Лаборатории Касперского»**.
4. Выберите **Все задачи** → **Очистить хранилище обновлений**. Дождитесь, пока закончится очистка.

![[Pasted image 20260127134451.png]]5. Перезапустите службу Сервера администрирования:

```cmd
net stop kladminserver   
net start kladminserver
```

6. Откройте Kaspersky Security Center и перейдите в раздел **Задачи**.
7. Откройте контекстное меню задачи **Загрузка обновлений в хранилище Сервера администрирования** и выберите **Запустить**.  
    Если задачи нет в списке, создайте новую по [инструкции](https://click.kaspersky.com/?hl=ru&link=helpupdatelinks_kb&pid=ksc15&version=1.0.0&helpid=180697 "Создание задачи Загрузка обновлений в хранилище Сервера администрирования").
