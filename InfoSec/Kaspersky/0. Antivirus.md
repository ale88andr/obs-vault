- **Вредоносное ПО (malware)** — программное обеспечение, намеренно разработанное для нарушения нормальной работы компьютеров, получения несанкционированного доступа, сбора конфиденциальной информации или нанесения другого ущерба .
- Основные признаки: **скрытность**, **автоматическое распространение**, **выполнение нежелательных операций**.

## Основные классы вредоносного ПО

| Тип                  | Характеристика                                                                                                                                                                                                                                                             | Примеры                                                                                        |
| -------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------- |
| **Вирус**            | Самореплицируется, внедряется в другие программы или файлы. Активируется при запуске заражённого ПО.                                                                                                                                                                       | Brain, Melissa                                                                                 |
| **Червь**            | Самостоятельно распространяется по сети, используя уязвимости ОС. Не требует действий пользователя.                                                                                                                                                                        | Morris Worm (1988)                                                                             |
| **Троян**            | Маскируется под легитимное ПО, активируется пользователем. Даёт злоумышленнику доступ к системе.                                                                                                                                                                           | Zeus Trojan                                                                                    |
| **Шифровальщик**     | Блокирует или шифрует данные, требует выкуп.                                                                                                                                                                                                                               | WannaCry, Locky                                                                                |
| **Spyware**          | Скрытно собирает данные о пользователе (пароли, сайты, клавиши).                                                                                                                                                                                                           | CoolWebSearch                                                                                  |
| **Кейлогер**         | Записывает нажатия клавиш и движения мыши.                                                                                                                                                                                                                                 | Keylogger.Win32                                                                                |
| **Руткит**           | Скрывает присутствие других вредоносных программ, обеспечивает скрытый доступ.                                                                                                                                                                                             | Sony Rootkit (2005)                                                                            |
| **Fileless malware** | Особый класс вредоносного ПО, который не использует традиционные исполняемые файлы для заражения системы. Вместо этого он работает напрямую в памяти или злоупотребляет встроенными инструментами операционной системы.                                                    |                                                                                                |
| **Bots**             | Программы, которые автоматически выполняют задачи в сети. Они могут быть как легитимными (например, чат-боты или поисковые краулеры), так и вредоносными, когда используются злоумышленниками для атак.                                                                    | - **Спам-боты** <br>- **DDoS-боты**<br>- **Клик-боты**<br>- **Кража данных**<br>- **IoT-боты** |
| **Spear phishing**   | Целевая разновидность фишинга, когда злоумышленники отправляют персонализированные сообщения конкретным людям или организациям, чтобы повысить вероятность успеха атаки. В отличие от массового фишинга, такие атаки тщательно готовятся и используют информацию о жертве. |                                                                                                |
