Для работы **PKI Client 1.6.1** c `WebUnit` нужно чтобы было установлено обновление `KB3140245`. Также в реестре поправить ветку в `Schannel` c `TLS 1.2`, создав раздел `Server`:

1. Откройте редактор реестра (`regedit.exe`).
2. Перейдите в раздел `HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\SecurityProviders\SCHANNEL\Protocols\TLS 1.2\Server`. Если его нет - создайте.
3. Создайте или измените значение `Enabled` `DWORD` и установите для него значение `1`, чтобы включить `TLS 1.2`. 
4. Создайте или измените значение `DisabledByDefault` `DWORD` и установите для него значение `0`, чтобы сервер мог согласовывать протокол.
5. Перезагрузить компьютер.
