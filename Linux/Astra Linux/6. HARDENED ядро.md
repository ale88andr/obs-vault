либо, через `cli`

для включения:

```Shell
sudo astra-nochmodx-lock enable
```

для выключения:

```Shell
sudo astra-nochmodx-lock disable
```

Для проверки состояния запрета установки бита исполнения

```Shell
cat /parsecfs/nochmodx
```

`0` - выключен, `1` - включен
