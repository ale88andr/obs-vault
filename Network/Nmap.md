Простое сканирование, с `1` по `1000` порт

```
nmap 192.168.0.1
```

Указываем диапазон портов и IP-адресов (`nmap -p 80`, `nmap -p "*"`)

```
nmap -p 100-25525 192.168.0.1-10
```

Cканируем топовые порты

```
nmap --top-ports 10 192.168.0.1
```

Cканируем без пинга

```
nmap -Pn 192.168.0.1
```

Определение ОС

```
nmap -O 192.168.0.1
```

Пытаемся узнать, что за сервисы запущены на открытых портах

```
nmap -sV 192.168.0.1
```

Пинг-сканирование

```
nmap –sP 192.168.0/24
```

Ищем активные хосты

```
nmap -sn 192.168.1.0/24
```

nmap -sO 192.168.0.1 - определяем поддерживаемые протоколы

nmap -sT 192.168.0.1 - сканируем все TCP-порты

nmap -p T:80 192.168.0.1 - определенный TCP-порт

nmap -sU 192.168.0.1 - сканируем UDP-порты (nmap -p U:53)

nmap -sA 192.168.0.1 - определяем, есть ли файервол

nmap -sS 192.168.0.1 - скрытое сканирование
