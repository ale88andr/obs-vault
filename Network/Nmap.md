Простое сканирование, с `1` по `1000` порт

```
nmap 192.168.0.1
```

Указываем диапазон портов и IP-адресов (`nmap -p 80`, `nmap -p "*"`)

```
nmap -p 100-25525 192.168.0.1-10
```

Cканируем топовые порты

```
nmap --top-ports 10 192.168.0.1
```

Cканируем без пинга

```
nmap -Pn 192.168.0.1
```

Определение ОС

```
nmap -O 192.168.0.1
```

Пытаемся узнать, что за сервисы запущены на открытых портах

```
nmap -sV 192.168.0.1
```

Пинг-сканирование

```
nmap –sP 192.168.0/24
```

Ищем активные хосты

```
nmap -sn 192.168.1.0/24
```

Определяем поддерживаемые протоколы

```
nmap -sO 192.168.0.1
```

Cканируем все TCP-порты

```
nmap -sT 192.168.0.1
```

Определенный TCP-порт

```
nmap -p T:80 192.168.0.1
```

Сканируем UDP-порты (`nmap -p U:53`)

```
nmap -sU 192.168.0.1
```

Определяем, есть ли файервол

```
nmap -sA 192.168.0.1
```

Скрытое сканирование

```
nmap -sS 192.168.0.1
```
